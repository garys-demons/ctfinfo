# CTF INFO

A mobile app that keep you updated about latest CTFs and its resources (Under development)

## Features

- View Upcoming CTFs
- View CTF details
- View top teams
- View team details
- CTF Writeups (Coming Soon)
- Your team details (Coming Soon)
- Notification about upcoming CTFs (Coming Soon)
